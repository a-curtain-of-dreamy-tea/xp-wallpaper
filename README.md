**小胖壁纸-微信小程序** 

小胖壁纸 是一个 基于 uni-app 开发的微信小程序，使用Django Admin 搭建后端服务；可配置微信流量主；

 **需要修改内容** 

1. 小程序key需要修改为自己的：在manifest.json中mp-weixin-appid
2. 接口配置地址需要修改为自己的：
    common->api->index 中 basehost，对应的接口调用地址需要按照自己的接口地址进行修改；
